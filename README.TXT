Explanation:
Simple AS3 class to see if the Android device features like Camera, GPS, Geolocation etc. are available on the device/phone.

Usage:
DeviceSupport.showSupportedFeatures();

For more information on the use of this Class and available methods, take a look at: http://www.funky-monkey.nl/blog/?p=632
